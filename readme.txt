== Description ==

This plugin allows you to list the most popular posts and pages of your blog according to your Google Analytics statistics.

For this plugin to work you will need to install Google Analytics Dashboard first http://wordpress.org/extend/plugins/google-analytics-dashboard/

*	Option do display Top X posts/pages
** Last week
** This week
** Last month
** This month 
*	Nofollow on/off on the links
*	You can add onclick events if you wish to add extra event tracking.
*	You can specify pages and posts that the widget should ignore.
*	You can trim the titles of your links in case you wish to remove the redundant name-of-your-blog-string

== Installation ==

1. Download and install Google Analytics Dashboard http://wordpress.org/extend/plugins/google-analytics-dashboard/
2. Upload the folder `sz-google-analytics-widget` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Add the widget to your sidebar
5. Configure the widget according to your wishes
6. Done!
